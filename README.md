# Developer Landing

![Vista previa del proyecto](./preview.png)

Landing page interactiva desarrollada con **HTML**, **CSS** y **JavaScript**, con una estética moderna y efectos visuales construidos sobre **Canvas API**, que es el elemento principal y más destacado del proyecto.

## Descripción

Este proyecto es una **landing page visual e interactiva** creada para presentar una identidad profesional de forma atractiva, moderna y dinámica.

A diferencia de una landing convencional, esta propuesta pone el foco en la experiencia visual, utilizando **Canvas** para generar partículas, movimiento reactivo al cursor y efectos de clic en tiempo real sobre el nombre principal. El resultado es una pantalla de presentación liviana, llamativa y con un estilo tecnológico.

## Lo más destacado del proyecto

- Animación principal desarrollada con **Canvas API**
- Partículas que forman el nombre principal
- Interacción en tiempo real al mover el mouse
- Efecto visual de expansión al hacer clic
- Fondo con atmósfera tecnológica
- Diseño centrado, limpio y moderno
- Íconos sociales con enlaces externos
- Favicon personalizado
- Estructura simple, rápida y liviana

## Tecnologías utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**
- **Canvas API**

## Objetivo

El objetivo de este proyecto fue crear una landing page enfocada en lo visual, explorando animaciones interactivas y efectos dinámicos en el navegador, sin depender de frameworks externos.

También funciona como una práctica para destacar el uso de **Canvas** en interfaces modernas, combinando diseño, interacción y rendimiento en una sola pantalla.
